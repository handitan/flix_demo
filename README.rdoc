== README

This is my first Rails 4 application, which allows users to update movies information
and their review.
Project timeline: one week.

This was done as part of Pragmatic Studio's "Rails Programming" course:
http://pragmaticstudio.com/rails

Requirements:
* Ruby 2.0.0
* Rails 4.0.0

To prepopulate the Database, please do the following:
<tt>rake db:reset</tt>
